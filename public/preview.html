How it works

Capture → Organize → Summarize

Every output is grounded. If a summary line can’t cite a note, it doesn’t ship.
01

A single observation, timestamped, with optional context (subject/activity).

Privacy

Built to be school-safe.

Clear boundaries: summarize and label teacher notes only. No hidden inference. No vague claims.

Citations-first: every AI claim links to source notes.
Access control: roles, audit logs, and exports.
Data boundaries: only selected notes are processed.
